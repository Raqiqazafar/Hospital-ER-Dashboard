# 🏥 Hospital Emergency Room Dashboard

An interactive Excel dashboard for analyzing hospital ER operations, built to improve efficiency and support data-driven decisions.

## 📊 KPIs Tracked
- **No. of Patients** — daily count with area sparkline
- **Average Wait Time** — in minutes with trend sparkline
- **Patient Satisfaction Score** — daily average with trend

## 📈 Charts Included
- Patient Admission Status (Admitted vs Not Admitted)
- Patient Age Distribution (0–79 age groups)
- Timeliness — % seen within 30 minutes
- Gender Analysis
- Department Referrals

## 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)

## 📅 Data Range
- April 2023 – March 2024 (Monthly filter available)
